# Float

Practitioner-led clinical platform for CBT-based anxiety treatment.

See ARCHITECTURE.md for full technical documentation.

## Structure

- `apps/web` — Practitioner and parent dashboard (React)
- `apps/mobile` — Teen experience (React Native + Expo)
- `packages/api-client` — Shared API client and types
- `packages/clinical-logic` — Shared clinical data logic
- `packages/ui-tokens` — Shared design tokens
- `backend` — FastAPI backend
